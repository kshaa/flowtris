# HTML5 / WebRTC Tetris
Motivation for this project was a lack of good multiplayer tetris
written with nice controls and without flash as a dependency.

# Tech stack  
* Koa
* React
* Redux
* SimpleWebRTC
* Webpack
* [Custom tetris engine](https://github.com/kshaa/tetris-engine)

# Compiling
```
    npm install
    npm run build
```
